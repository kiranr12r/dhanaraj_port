# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Dhanu37400/dhanaraj-portfolio.git
```

Go to the project directory

```bash
cd dhanaraj-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
